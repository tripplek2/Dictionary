## Dictionary App
This is a simple dictionary that enable users to search for a word and fetch information using an API.

## Features
1. Search for a word.
2. Display definitions.
3. fetch data from an API.

## Technologies used
1. HTML
2. CSS
3. Javascript(Fetch API)

## Functionality
1. User types a word on the search bar.
2. form submission is captured using JS.
3. App sends information to the API.
4. Response is processed and displayed on the page.
